## Stack

- Vite 8 + React 19 + TypeScript
- Tailwind CSS 4 (NEVER use `@apply` — Tailwind 4 doesn't support it in `@apply` context; always inline classes)
- bun (NOT npm/yarn) — `bun install`, `bun dev`, `bun run build`, `bun scripts/X.ts`

- canvas-confetti for celebrations
- Web Audio API for sounds (programmatic, no asset files)

## Critical rules

- **NEVER run `bun run dev` automatically** — user runs dev server themselves; killing it disrupts workflow
- **Slovak grammar matters** — user is native Slovak speaker and WILL catch errors. Especially:
  - vzor "chlap" plural is `chlapi` (mäkké i, soft consonant)
  - "učiteľ" → "učitelia" doesn't fit "chlap" cleanly; use "žiak" (žiaci) for clean examples
  - Gender-aware: "pripravená/pripravený", "trénovala/trénoval", "šikovná/šikovný" — use `gen()` helper from App.tsx
  - "vaucher" is masc. inanimate → accusative is "vaucher" not "vauchera"
- **NEVER expose parenting strategy/philosophy in UI copy** — "rewards are rewards", no "this is your inner reward" lecturing
- **OpenAI image model:** use exactly `gpt-image-2-2026-04-21` (user explicitly chose this version)
- **Don't lecture** — keep copy brief, kid-friendly, fun
- **Mobile-first** — every UI change must work on mobile, sticky bottom CTAs, ≥44px tap targets, `100svh`
- **Limit widgets on home** — user explicitly hates clutter; consolidate stats into single hero card

## Conventions

- Slovak quotes: „" (German-style)
- Kid-friendly emoji liberally
- Theme via CSS vars `[data-theme="boy"|"girl"]` on body — NEVER hard-code pink/cyan, use `var(--grad-from)` etc.
- Question type: `'choice'` (with choices array, answer = id) or `'text'` (with `acceptable` for variants like comma vs. dot decimals)
- IDs: math packs `pm1-N`/`pm2-N`, slovak `ps1-N`/`ps2-N`, base m1-30/s1-30
- Storage keys: `prijimacky-game-v1`, `prijimacky-seen-rewards`, `prijimacky-sound-v1`

## Game design (calibrated)

- **Acceptance odds base rate: 15 %** (top BA gymnasiums Gamča/J. Hronec/etc accept ~25 from ~250)
- **Pace: 90 sec/q in tests** ("hard in training, easy in life" — real exam is 103-180 sec/q)
- **Quiz modes:** Tréning (5q, no timer, immediate feedback) / Malý test (10q, 15min, defer feedback) / Veľký test (20q, 30min, defer + skip)
- **Defer mode:** answers hidden during quiz → review screen at end → final submit
- **Vouchers ONLY at L5/10/15/20** (every 5th level) — milestones at other levels are intrinsic affirmations only
- **Mastery:** question leaves review pool after 2× consecutive correct answers
- **XP:** 10/correct + 10 (streak 3+) or 25 (streak 5+) + 50 (perfect)

## Data persistence quirk

PWA installed-app context can have separate localStorage from browser context on iOS.
**Recommendation in README:** install PWA BEFORE onboarding, so progress doesn't reset.

## What user dislikes (learned)

- Cluttered UI — wants 1 consolidated stats card not 5 separate widgets
- Filozofovanie v UI ("inner value", "no one can take this from you") — wants concise rewards
- Ungrammatical Slovak — has caught: tvrdé/mäkké i mixup, "vauchera" wrong accusative, "učiteľ" as "chlap" example
- Hard questions for kid — daughter prefers easier; consider difficulty tagging in future iteration
